# Mindmap Editor

### Project Structure

- app
- mm-render

### 1. Install Dependencies

This project uses NodeJS to run the app. 
First download and install NodeJS (https://nodejs.org). 

```bash
cd app
npm i
```

### 2. Build
```bash
npm run build
```

### 3. Run

```bash
node server.js
```
Go to: http://localhost:3000/
